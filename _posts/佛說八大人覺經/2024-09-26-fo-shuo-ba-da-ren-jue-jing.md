後漢安息國三藏安世高譯

為佛弟子常於晝夜，至心誦念八大人覺。

第一*<b>覺悟</b>*：
: - *<b>世間無常</b>，<b>國土危脆</b>*。
    - *<b>四大<i>苦空</i></b>，<b><i>五陰</i><mark>無</mark><i>我</i></b>*。
	  - *<b>生滅變異</b>，<b>虛偽<i>無主</i></b>*。
	    - *<b><i>心</i>是惡源</b>，<b><i>形</i>為罪藪</b>*。
		  - *<b><i>如是</i><mark>觀察</mark></b>，<b>漸離</b>生死*。

第二*<b>覺知</b>*：
: - *<b>多欲為苦</b>*。
    - *<b>生死、疲勞</b>，<b><mark>從</mark><i>貪欲</i><mark>起</mark></b>*。
	  - *<b>少欲<i>無為</i></b>，<b><i>身心</i><mark>自在</mark></b>*。

第三*<b>覺知</b>*：
: - *<b><i>心</i><mark>無</mark><i>厭足</i></b>*。
    - *<b>唯得<i>多求</i></b>，<b><mark>增長</mark><i>罪惡</i></b>*。
	  - *菩薩不爾*{:.red}。
	    - *<b>常念知足，安貧<mark>守</mark><i>道</i>，唯<i>慧</i>是業</b>*{:.red}。

第四*<b>覺知</b>*：
: - *懈怠<b>墜落</b>*。
    - *<b>常<mark>行</mark><i>精進</i></b>，<b><mark>破</mark><i>煩惱惡</i></b>，<b>摧伏<i>四魔</i></b>，<b><mark>出</mark><i>陰界獄</i></b>*。

第五*<b>覺悟</b>*：
: - *愚癡<b>生死</b>*。
    - *菩薩<b>常念</b>，<b><i>廣</i><mark>學</mark><i>多</i><mark>聞</mark></b>，<b>增長<i>智慧</i></b>，<b>成就<i>辯才</i></b>，<b><mark>教化</mark><i>一切</i></b>*。
	  - *<b><i>悉</i>以大樂</b>*。

第六*<b>覺知</b>*：
: - *<b>貧苦<mark>多</mark><i>怨</i></b>，橫<b><mark>結</mark><i>惡緣</i></b>*{:.red}。
    - *菩薩<b>布施</b>，<b><mark>等</mark><i>念</i>冤親</b>，<b>不念<i>舊惡</i></b>，<b>不憎<i>惡人</i></b>*。

第七*<b>覺悟</b>*：
: - *<b><i>五欲</i>過患</b>*。
    - *<b>雖<mark>為</mark><i>俗人</i></b>，<b>不染<i>世樂</i></b>*。
	  - *<b><mark>常念</mark>三衣</b>，<b>瓦鉢法器</b>*。
	    - *<b><i>志願</i>出家</b>，<b><mark>守</mark><i>道</i>清白</b>*。
		  - *<b><i>梵行</i><mark>高遠</mark></b>，<b>慈悲一切</b>*。

第八*<b>覺知</b>*：
: - *生死熾然，<b><i>苦惱</i>無量</b>*。
    - *<b>發<i>大乘心</i></b>，<b>普濟一切</b>*。
	  - *願<b>代</b>眾生，<b>受</b>無量苦*。
	    - *<b>令</b>諸眾生，<b>畢竟<i>大樂</i></b>*。

如此*<b>八事</b>*。
: - *<b>乃<mark>是</mark><i>諸佛菩薩大人</i>之<mark>所覺悟</mark></b>*。
    - *<b>精進<mark>行</mark><i>道</i></b>，<b>慈悲<mark>修</mark><i>慧</i></b>，<b><mark>乘</mark><i>法身船</i></b>，<b><mark>至</mark><i>涅槃岸</i></b>*。
	  - *復還生死，度脫眾生*。
	    - *<b>以</b>前八事，<b>開導</b>一切*。
		  - *令諸眾生，<b>覺</b>生死苦，<b>捨離<i>五欲</i></b>，<b><mark>修</mark><i>心聖道</i></b>*。

若佛弟子，誦此八事。
: - *<b>於<i>念念中</i></b>，<b><mark>滅</mark>無量罪</b>*。
    - 進趣菩提，速登正覺。
	  - 永斷生死，常住快樂。
